# Clean YouTube Reysu (Text Only)

Userscript для мобильного YouTube: убирает **все превью** (на главной, в
трансляциях, играх, новостях, спорте, обучении, поиске, разделах — везде),
оставляет только текст и аватарки каналов, добавляет аккуратные серые
разделители под цвет темы.

## Что делает

- 🚫 Прячет все превью видео в любом разделе (оба движка YouTube)
- 👤 Оставляет аватарки каналов (36px, кружком)
- ➖ Тонкие разделители между карточками под цвет темы (тёмная/светлая)
- 📣 Убирает рекламу, Shorts, ленту сообщества и рекомендации под видео

## Установка

1. Поставь менеджер userscript'ов — [Tampermonkey](https://www.tampermonkey.net/)
   или [Violentmonkey](https://violentmonkey.github.io/).
2. Открой файл [`youtube.user.js`](youtube.user.js) → менеджер предложит установить.
3. Перезагрузи `m.youtube.com`.

## Настройки (вкл/выкл функций)

В начале `youtube.user.js` есть блок `CONFIG` — поставь `false` у любой
функции, чтобы её выключить (превью, аватарки, разделители, Shorts, реклама,
сообщество, «Ещё темы», удалённые разделы, плейлисты, аватарки истории,
автозапуск). После изменения перезагрузи YouTube.

## Обновление

Скрипт самодостаточный — просто замени `youtube.user.js` в менеджере новой
версией (текущая — `2.0`) и перезагрузи страницу.
